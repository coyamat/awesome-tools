# 作業ログ出力コマンド

これまでの会話で行った作業の手順と結果をMarkdownファイルに書き出してください。

## 出力ファイル
- ファイル名: `<YYYYMMDD>-<作業内容のサマリ>.md`
  - 例: `20260308-add-dark-mode.md`
- 出力先: カレントディレクトリ

## ファイル形式

```markdown
# <作業内容のタイトル>

日付: YYYY-MM-DD

## 概要
<作業の概要を1-2文で>

## 作業内容

### 1. <ステップ1のタイトル>
- 実施した内容
- 変更したファイル（あれば）

### 2. <ステップ2のタイトル>
...

## 変更ファイル一覧
- `path/to/file1.ts` - 変更内容の説明
- `path/to/file2.ts` - 変更内容の説明

## 結果
<最終的な結果や確認事項>

## 備考
<補足情報があれば>
```
